# 常用参数讲解

## 快速开始

推荐先使用严格观察池：

```bash
python3 low_absorb_screener.py \
  --min-market-cap-yi 800 \
  --lookback-days 240 \
  --recent-days 10 \
  --min-hit-days 3 \
  --lowest-down-percent 5 \
  --min-avg-amount-yi 3 \
  --recent-volume-shrink-ratio 0.6 \
  --max-20d-drop-pct 15 \
  --min-distance-from-60d-low-pct 5 \
  --max-latest-volume-percentile 20 \
  --require-not-break-low-volume-low \
  --min-low-absorb-score 70 \
  --workers 3
```

---

## 核心参数速查表

### 地量识别（必选）

| 参数 | 含义 | 常用值 | 说明 |
|---|---|---|---|
| `--lookback-days` | 历史统计周期 | 180 / 240 | 180 偏灵敏，240 更稳健 |
| `--recent-days` | 最近观察窗口 | 7 / 10 | 窗口越小越灵敏 |
| `--min-hit-days` | 至少命中几天 | 2 / 3 | 宽松用 2，严格用 3 |
| `--lowest-down-percent` | 下跌日成交量最低百分位 | 5 / 10 | 5 更严格，10 更宽松 |

### 流动性与缩量（推荐）

| 参数 | 含义 | 常用值 | 说明 |
|---|---|---|---|
| `--min-avg-amount-yi` | 近 20 日平均成交额下限（亿） | 2 / 3 / 5 | 过滤无人问津的票 |
| `--recent-volume-shrink-ratio` | 近 5 日 / 60 日均量上限 | 0.6 / 0.7 | 越低越要求整体缩量 |
| `--max-latest-volume-percentile` | 最新成交量分位上限 | 20 / 30 | 越低越要求当前仍缩量 |

### 破位保护（推荐）

| 参数 | 含义 | 常用值 | 说明 |
|---|---|---|---|
| `--max-20d-drop-pct` | 近 20 日最大回撤（%） | 15 / 20 | 超过则剔除 |
| `--min-distance-from-60d-low-pct` | 距 60 日低点最小距离（%） | 3 / 5 | 贴近新低则剔除 |
| `--require-not-break-low-volume-low` | 不跌破地量日低点 | 加上 | 严格版强烈建议 |

### 均线确认（可选）

| 参数 | 含义 | 严格程度 |
|---|---|---|
| `--require-price-above-ma5` | 站上 5 日线 | 低 |
| `--require-price-above-ma10` | 站上 10 日线 | 中 |
| `--require-price-above-ma20` | 站上 20 日线 | 高 |

观察池不加；确认池加 MA5；保守加 MA10 或 MA20。

### 信号与评分（可选）

| 参数 | 含义 | 常用值 | 说明 |
|---|---|---|---|
| `--require-confirm-signal` | 要求放量阳线确认 | 加上 | 很严格，会大幅减少结果 |
| `--min-reward-risk-ratio` | 最低盈亏比 | 1.5 / 2 | 越高越严格 |
| `--min-low-absorb-score` | 最低综合评分 | 60 / 70 / 75 | 越高越严格 |
| `--min-market-cap-yi` | 市值下限（亿） | 500 / 800 / 1000 | 越高越聚焦大盘股 |

## 调参原则

### 结果太多 → 加严

```bash
--min-market-cap-yi 1000          # 提高市值门槛
--lowest-down-percent 5           # 更严格地量标准
--min-hit-days 3                  # 要求更多天命中
--min-avg-amount-yi 5             # 提高成交额门槛
--recent-volume-shrink-ratio 0.6  # 更低的缩量比
--max-latest-volume-percentile 15 # 更低的当前量分位
--require-not-break-low-volume-low # 加上地量保护
--require-price-above-ma5         # 加上均线确认
--min-low-absorb-score 75         # 提高评分门槛
```

### 结果太少 → 放宽

```bash
--min-market-cap-yi 500           # 降低市值门槛
--lowest-down-percent 10          # 放宽地量标准
--min-hit-days 2                  # 减少命中天数
--min-avg-amount-yi 2             # 降低成交额门槛
--recent-volume-shrink-ratio 0.7  # 放宽缩量比
--max-latest-volume-percentile 30 # 放宽当前量分位
--min-low-absorb-score 60         # 降低评分门槛
```

## 参数组合示例

### 平衡观察池（日常使用）

```bash
--min-market-cap-yi 500 --lowest-down-percent 10 --min-hit-days 2 \
--min-avg-amount-yi 2 --recent-volume-shrink-ratio 0.7 \
--max-latest-volume-percentile 30
```

### 严格观察池（结果太多时）

```bash
--min-market-cap-yi 800 --lowest-down-percent 5 --min-hit-days 3 \
--min-avg-amount-yi 3 --recent-volume-shrink-ratio 0.6 \
--max-latest-volume-percentile 20 --require-not-break-low-volume-low
```

### 确认型低吸（找买点）

```bash
--min-market-cap-yi 800 --lowest-down-percent 5 --min-hit-days 3 \
--min-avg-amount-yi 3 --recent-volume-shrink-ratio 0.65 \
--max-latest-volume-percentile 25 --require-not-break-low-volume-low \
--require-price-above-ma5 --require-confirm-signal \
--min-reward-risk-ratio 2 --min-low-absorb-score 75
```

## 其他常用参数

| 参数 | 含义 | 使用场景 |
|---|---|---|
| `--refresh-candidates` | 强制刷新候选表 | 改了市值门槛或缓存过旧时 |
| `--skip-kline-update` | 只用本地 K 线 | 离线调试时 |
| `--force-kline-update` | 重新拉取所有 K 线 | 数据异常需重建时 |
| `--force-renormalize` | 强制成交量单位归一化 | 发现成交量数据异常时 |
| `--exclude-today` | 不包含今日数据 | 盘中运行时 |
| `--workers 3` | 并发线程数 | 网络慢时降低，快时提高 |
