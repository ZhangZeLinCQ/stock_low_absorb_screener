# 常用参数讲解

推荐先使用：

```bash
python3 low_absorb_screener.py \
  --min-market-cap-yi 800 \
  --lookback-days 240 \
  --recent-days 10 \
  --min-hit-days 3 \
  --lowest-down-percent 5 \
  --min-avg-amount-yi 3 \
  --recent-volume-shrink-ratio 0.6 \
  --max-20d-drop-pct 15 \
  --min-distance-from-60d-low-pct 5 \
  --max-latest-volume-percentile 20 \
  --require-not-break-low-volume-low \
  --min-low-absorb-score 70 \
  --workers 3
```

## 关键参数

| 参数 | 含义 | 常用值 |
|---|---|---|
| `--min-market-cap-yi` | 市值下限，单位亿元 | 500 / 800 / 1000 |
| `--lookback-days` | 历史统计周期 | 180 / 240 |
| `--recent-days` | 最近观察窗口 | 7 / 10 |
| `--min-hit-days` | 最近窗口内至少命中几天 | 1 / 2 / 3 |
| `--lowest-down-percent` | 下跌日成交量进入历史最低百分之几 | 5 / 10 |
| `--max-latest-volume-percentile` | 最新成交量处于历史最低百分之几以内 | 20 / 30 |
| `--min-avg-amount-yi` | 近 20 日平均成交额下限，单位亿元 | 2 / 3 / 5 |
| `--recent-volume-shrink-ratio` | 近 5 日均量 / 近 60 日均量上限 | 0.6 / 0.7 |
| `--max-20d-drop-pct` | 近 20 日最大允许回撤 | 15 / 20 |
| `--min-distance-from-60d-low-pct` | 距 60 日低点最小距离 | 3 / 5 |
| `--require-not-break-low-volume-low` | 要求不跌破最近地量日低点 | 严格版建议加 |
| `--require-price-above-ma5` | 要求站上 5 日线 | 确认型加 |
| `--require-confirm-signal` | 要求放量阳线确认 | 很严格 |
| `--min-reward-risk-ratio` | 最低盈亏比 | 1.5 / 2 |
| `--min-low-absorb-score` | 最低低吸综合评分 | 60 / 70 / 75 |

## 调参原则

结果太多：提高市值、降低 `--lowest-down-percent`、增加 `--min-hit-days`、提高评分门槛。

结果太少：降低市值、放宽到 `--lowest-down-percent 10`、去掉 `--require-confirm-signal`、降低评分门槛。
